# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Report violations to `hlopescau@gmail.com` or via a private [security advisory](https://github.com/HugoLopes45/git-memory/security/advisories/new). All reports are reviewed promptly and the reporter's identity is kept private.
